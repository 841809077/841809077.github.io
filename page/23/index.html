开发ambari API

2018-06-04Ambari

声明：博主写了一些Ambari系列文章，可以在历史文章中查看。

一、概述

  1. 如果要开发ambari API的话，我们需要在ambari-server的文件里面添加相关java代码。

  2. 然后整体编译ambari，可以不编译ambari-metrics系列。

  3. 编译成功后，yum remove ambari-server

  4. yum install ambari-server-*.rpm

  5. ambari-server setup

  6. vim /etc/ambari-server/conf/ambari.properties

  7. server.jdbc.driver.path=/usr/share/java/mysql-jdbc.driver.jar

    mysql-jdbc.driver.jar下载地址 将jar包放入/usr/share/java/目录下

  8. ambari-server setup –jdbc-db=mysql –jdbc-driver=/usr/share/java/mysql-jdbc.driver.jar

  9. ambari-server start

  10. 访问ambari页面
